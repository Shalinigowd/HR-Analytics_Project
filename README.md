# AI-Powered HR Analytics Dashboard with Attrition Prediction

##  Project Overview
This project analyzes employee workforce data and attrition trends using Power BI and Machine Learning techniques. 

The dashboard provides interactive visualizations, department-wise analysis, AI insights, and predictive analytics to help organizations understand employee behavior and improve retention strategies.

---

# Objectives

- Analyze employee attrition trends
- Identify factors affecting employee resignation
- Perform department-wise workforce analysis
- Build an interactive HR analytics dashboard
- Predict employee attrition using Machine Learning
- Generate business recommendations from insights

# 📌 Key Insights

- Sales department has highest attrition rate
- Employees working overtime are more likely to leave
- Low job satisfaction increases resignation probability
- Younger employees tend to change jobs more frequently
- Employee engagement impacts retention significantly

---

# 💡 Business Recommendations

- Improve employee work-life balance
- Reduce overtime workload
- Conduct employee engagement programs
- Provide career growth opportunities
- Use predictive analytics for HR decision-making

---

# 🧠 Machine Learning Model

The Random Forest algorithm was used to predict employee attrition because it provides:
- High accuracy
- Better performance
- Reduced overfitting
- Strong feature importance analysis
